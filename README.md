# MurinTranslator

Десктопное приложение на JavaFX для двустороннего перевода текста между русским и вымышленным "муринским" языком.

## Описание

Муринский язык — это интернет-диалект, основанный на намеренном искажении слов, имитации опечаток и специфической суффиксации. Это приложение автоматизирует процесс перевода, используя комбинацию словарных замен и алгоритмической генерации.

## Функционал

* **Двусторонний перевод:**
    * Русский → Муринский
    * Муринский → Русский
* **Словарные замены:** Обработка специфических слов (например, `друг` → `друн`, `дед` → `дод`) с учетом падежей.
* **Алгоритмическая обработка:** Автоматическое добавление суффикса `-ность` для слов, отсутствующих в словаре.
* **Умные исключения:**
    * Игнорирование местоимений (*я, ты, мы, свой, мой* и т.д.).
    * Игнорирование служебных частей речи (предлоги, союзы).
    * Сохранение структуры предложения и знаков препинания.
* **Удобный интерфейс:** Кнопка быстрой смены направления перевода (Swap).

## Технологии

* **Язык:** Java 17+
* **UI Фреймворк:** JavaFX

## Структура проекта

* `src/Main/MurinTranslatorApp.java` — Точка входа и графический интерфейс.
* `src/Main/TranslationService.java` — Логика перевода и алгоритмы.
* `src/Main/DictionaryRepository.java` — База данных слов, падежей и исключений.

## Правила "Муринского"

Алгоритм основан на следующих принципах:
1.  **Жесткие замены:**
    * *Друг* → *Друн*
    * *Враг* → *Вран*
    * *Мама* → *Мом*
    * *Сын* → *Сыр*
2.  **Суффиксация:** Если слово не является исключением, у него отсекается окончание и добавляется `-ность` (например, *Кот* → *Котность*).
3.  **Исключения:** Слова вроде *фог*, *что*, а также местоимения остаются без изменений.
